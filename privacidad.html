Aviso de Privacidad

Este aviso describe cómo Idiomas Avanza recopila y protege sus datos personales, conforme a la Ley Federal de Protección de Datos Personales en Posesión de Particulares.

Responsable

Idiomas Avanza, con domicilio en Morelos 21BIS, Col. Apatlaco, CDMX. Contacto: contacto@idiomasavanza.mx.

Datos Recopilados

Se recopilan datos como nombre, correo, teléfono y datos de pago para fines académicos y administrativos.

Uso de Datos

La información será utilizada para la gestión académica, facturación, envío de comunicaciones y mejora de los servicios.

Protección de Datos

Se implementan medidas de seguridad para proteger los datos personales. No se transfieren datos a terceros salvo obligación legal.

Derechos ARCO

El usuario podrá solicitar acceso, rectificación, cancelación u oposición de sus datos personales enviando un correo a contacto@idiomasavanza.mx.

Cambios

Se podrán realizar actualizaciones a este aviso, las cuales serán publicadas en www.idiomasavanza.mx.
